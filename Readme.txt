NNApp (Newspaper Notification Application)

Назначение:
	Создание/редактирование рассылок-оповещений в СМИ о непоступивших номерах
	
Алгоритм работы:
	1) Строит список рассылок-оповещений из таблицы NespaperMail
	2) Строит список шаблонов сообщений из таблицы NespaperMailMessage
	3) Создание рассылки на основе списка СМИ из таблицы Newspaper
	4) Редактирование рассылки
	
Конфигурация:
	БД Identity - таблицы MVC5, используемые для аутентификации пользователя
	БД Main - остальные таблицы
	
Установка:
	Скопировать файлы в папку сайта IIS
